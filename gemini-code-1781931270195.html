특수학급 지원인력 주간 시간표

20년 차 특수교사의 노하우를 담은 매끄러운 통합학급 인력 지원 관리 시스템

💡 사용 방법:
1. 상단의 교사 이름 탭(A, B, C, D)을 눌러 해당 선생님의 시간표로 이동합니다.
2. 시간표 칸 내부를 클릭하여 내용을 바로 수정할 수 있습니다. (통합학급 학년-반, 학생 이름, 지원 내용 등 입력)
3. 변경 사항은 하단의 '현재 설정 저장하기' 버튼을 누르면 브라우저(LocalStorage)에 자동 저장되어, 다음번에 열어도 유지됩니다.
4. '현재 시간표 인쇄/PDF' 버튼을 누르면 해당 선생님의 시간표만 깔끔하게 출력하거나 PDF로 저장할 수 있습니다.
